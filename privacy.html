Privacy Policy

Last updated: May 19, 2026

Short version: critique collects no data. It runs entirely on your local machine. Nothing is sent anywhere.

What this plugin does

critique is a Claude Code plugin that injects a critical mindset into your AI sessions via local hooks and skill definitions. All processing happens inside Claude Code on your machine.

Data collection

critique does not collect, store, transmit, or share any personal data, usage data, telemetry, or analytics. There are no network requests, no tracking pixels, and no external services involved.

Local files

The plugin writes a temporary flag file (.critique-active) to your home directory to track session state. This file is read and written only by the plugin hooks running locally. It is never transmitted.

Third-party services

This website uses Google Fonts (Geist) loaded from Google's CDN. Google's privacy policy applies to that request.
